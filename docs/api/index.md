# API Reference

Welcome to the SciMath WASM API Reference. This library provides high-performance mathematical operations using WebAssembly.

## Modules

- [Basic Math](/api/basic) - Core mathematical functions and constants
- [Statistics](/api/stats) - Statistical analysis and probability distributions
- [Linear Algebra](/api/linalg) - Matrix and vector operations
- [Signal Processing](/api/signal) - FFT, filters, and convolution
- [Trigonometry](/api/trig) - Advanced trigonometric functions
- [Polynomials](/api/poly) - Polynomial arithmetic and root finding
- [Regression](/api/regression) - Curve fitting and data modeling
- [Complex Numbers](./complex.md) - Complex number arithmetic
- [Curve Fitting](./fitting.md) - Mathematical model fitting
- [Data Analysis](./analysis.md) - Signal processing and analysis
- [Calculus](/api/calculus) - Numerical integration and differentiation
- [Unit Conversions](/api/units) - Physical unit conversions
