# Unit Conversions

Utilities for converting between scientific units via `SciMathJS`.

## API Reference

### Temperature
- `celsiusToFahrenheit(c: number): number`
- `fahrenheitToCelsius(f: number): number`
- `celsiusToKelvin(c: number): number`

### Pressure
- `pascalToBar(pa: number): number`
- `barToPascal(bar: number): number`

### Distance
- `metersToInches(m: number): number`
